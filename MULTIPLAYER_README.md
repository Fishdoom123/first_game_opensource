# Neon Shooter - Мультиплеерный режим

## Быстрый старт

### 1. Запустить сервер мультиплеера
```bash
npm start
```
или
```bash
node server.js
```

Сервер будет запущен на **ws://localhost:8080**

### 2. Запустить клиент
```bash
python3 -m http.server 8000
```
Откройте **http://localhost:8000** в браузере

### 3. Выбрать мультиплеер в меню
- Нажмите кнопку **МУЛЬТИПЛЕЕР** на главном экране
- Нажмите **СОЗДАТЬ ИГРУ** или **ПРИСОЕДИНИТЬСЯ**

---

## Режимы игры

### 🤝 **Кооперативная (COOP)**
- **Все игроки** помогают друг другу против врагов
- Враги спавнятся для всех
- Максимум 4 игроков
- Урон и смерти общие

### ⚔️ **PvP (Боевой режим)**
- **Игроки против друг друга** (дружественный огонь включен)
- Враги тоже присутствуют, но слабее
- Максимум 4 игроков
- Каждый защищается отдельно

---

## Архитектура сетей

### Сервер (Node.js + WebSocket)
- **server.js** - основной сервер
- Управляет игровыми сессиями
- Синхронизирует позиции и выстрелы игроков
- Обрабатывает входящих/выходящих игроков

### Клиент (JavaScript)
- **network.js** - модуль сетевого взаимодействия
- **game.js** - интеграция мультиплеера
- **render.js** - отрисовка других игроков

## Коды ошибок

| Код | Значение |
|-----|----------|
| `CONNECT` | Подключение к серверу |
| `GAME_STATE` | Текущее состояние игры |
| `PLAYER_UPDATE` | Обновление позиции игрока |
| `PLAYER_JOINED` | Новый игрок присоединился |
| `PLAYER_LEFT` | Игрок покинул сессию |
| `ERROR` | Ошибка на сервере |

## Примеры использования API

### Создать игру
```javascript
networkManager.createGame('coop', 4);
```

### Присоединиться к игре
```javascript
networkManager.joinGame('game_abc123');
```

### Отправить позицию на сервер
```javascript
networkManager.updatePlayerPosition(player.x, player.y, player.angle);
```

### Отправить выстрел
```javascript
networkManager.shoot(x, y, vx, vy);
```

### Слушать событие обновления позиции
```javascript
networkManager.on('PLAYER_UPDATE', (data) => {
    console.log(`Player ${data.playerId} at (${data.x}, ${data.y})`);
});
```

## Развёртывание на сервер

### Для облачного сервера (Heroku, Replit, etc)
1. Установить Node.js
2. Загрузить файлы проекта
3. Запустить `npm install`
4. Установить переменную окружения `PORT`
5. Запустить `npm start`

### URL подключения для облака
```javascript
// Вместо localhost:8080 используйте:
const serverUrl = 'wss://your-domain.com/socket';
```

## Отладка

### Проверить подключение
```bash
# Сервер должен выдать:
# Сервер мультиплеера запущен на порту 8080
# Подключитесь к ws://localhost:8080

# В браузере консолью:
networkManager.isConnected // true = подключено
networkManager.gameId      // ID текущей игры
```

### Логирование
Все события логируются в консоль браузера и терминал сервера:
```
[Network] Подключено к серверу
[Network] Переподключение (попытка 1/5)...
```
